1: Co je JavaScript a k jakým účelům se používá?

Doplň vhodné odpovědi a využij k tomu nečíslovaného seznamu.

Použij v textu aspoň 3 internetové odkazy na zajímavé příklady využití JS!

JavaScript je skriptovací jazyk učený pro tvorbu moderních dynamických webů. Byl představen v devadesátých letech 20. století, jako reakce na klasické statické internetové stránky a otevřel prostor pro interaktivní webové aplikace i příjemnější uživatelské rozhraní doplněné animacemi a 2D i 3D grafikou.

JavaScript lze využít u zmíněných formulářů, ať už jako kontrolní prvek nebo při automatickém doplňování jednotlivých položek – ku příkladu jméno a příjmení. Další využití JS nalezne při asynchronní komunikaci, kde není nezbytně nutná interakce s uživatelem. S tím se pojí i událostmi řízená funkcionalita – animace po přejetí myší apod.

    5: Jaké operátory můžeme používat v JavaScriptu?

    Vypište operátory, které patří do uvedených skupin.

    U méně obvyklých operátorů dopište do závorky jejich význam!

    Matematické

    +, -, *, /, % (modulo - zbytek po celočíselném dělení), ** (umocnění)

    Přiřazovací

    =, +=, -=, *=, /=, %=

    Logické

    && (AND), || (OR), ! (NOT)

    Porovnávací

    == (rovná se - hodnota), != (nerovná se - hodnota), === (rovná se - hodnota i typ), !== (nerovná se - hodnota i typ), > (větší než), < (menší než), >= (větší nebo rovno), <= (menší nebo rovno)

    Bitové

    & (AND), | (OR), ^ (XOR), ~ (NOT), << (bitový posun vlevo), >> (bitový posun vpravo)

    Typové

    typeof (typ proměnné), instanceof (určuje, zda objekt je instancí určité třídy)

    /* Zobrazí popup okno s upozorněním */ alert('Nezadal jsi správné heslo!');

    prompt()

    /* Zobrazí popup okno s výzvou pro uživatele zadat text */ let password = prompt('Prosím, zadejte heslo:');
